# Machine-Learning-Project

Supervised machine learning that will differentiate between two emails authors( sara , chris) by having the words that each author mostly use and compare which one of three supervised machine learning algorithms is the most efficient one to do this task
This project has been made by the help of Udacity machine learning course and dataset 

### Prerequisites

This system depends on sklearn library

### Installing

open the terminal and install the libraries using pip

how to install Prerequisites Libraries
```
pip install sklearn 
```
